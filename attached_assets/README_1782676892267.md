# Handoff: FlipScan — eBay Visual Price Scanner

## Overview

FlipScan is a mobile-first web app for resellers and flippers (garage sales, thrift stores). The user points their phone camera at any item, the app performs a visual search against eBay, identifies the item, and returns:

- **Active listings**: current high, low, and average listed price + count
- **Sold comparisons**: 90-day high, low, average sold price + count
- **Sell-through rate**: sold ÷ (sold + active), expressed as a percentage with Hot / Moderate / Slow classification
- **Flip Analysis**: recommended buy-below price, suggested listing range, estimated profit

## Real API Integration Notes

This prototype uses hardcoded mock data. To wire up real data:

### 1. Visual Search / Item Identification
- **eBay Image Search API** (beta): `POST /buy/browse/v1/item_summary/search_by_image`
- Alternatively: **Google Vision API** → label detection → use result as eBay keyword search

### 2. Active Listings
- **eBay Browse API**: `GET /buy/browse/v1/item_summary/search?q={term}&filter=conditionIds:{used}`
- Extract: min price, max price, count, average (calculate from result set)

### 3. Sold Comparisons (90-day)
- **eBay Finding API** (legacy but still active): `findCompletedItems` with `SoldItemsOnly=true`
- Or **eBay Analytics / Marketplace Insights API** (requires higher API tier)

### 4. Sell-Through Rate
- Calculate as: `soldCount / (soldCount + activeCount)` over trailing 90 days
- eBay does not expose this directly — derive it from the two API calls above
